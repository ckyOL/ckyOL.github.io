2020

却说社畜以来，每日上班摧残没得精力写 blog，一不小心就到了 2020 年。2019 年只有一篇🌚，再不更给你们有关博🐦🐦之感。万万没想到 2020 年初疫情搞的假期越来越长，甚至出现远程上班。远程上班那可就放心 划水 休闲，于是有闲情更博。

换主题

本博先使用 indigo 主题已久，本来也是基本适用。前两日却有 gayhub 邮件轰炸，说什么 GitHub API Deprecation notice。于是便想更新主题，没想到主题年久失修，最后更新于去年 1 月。看来不能指望了，逛一圈发现新近出来的主题 Ayer 较为美观、功能齐全、更新迅速（昨天 PR gitalk 更新迅速 merged）。于是切换到 Ayer，本博更现代😆
